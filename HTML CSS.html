What is HTML?

HTML:Hyper Text Markup Language

HTML is the standard markup language for creating Web pages. It describes the structure of a Web page. It consists of a series of elements,which tells the browser how to display the content.

HTML can be assissted by technologies such as Cascading Style Sheets(CSS) and scripting languages such as JavaScript. HTML elements are building blocks of HTML pages. Its elements are delineated by tags,written using angle brackets. With HTML constructs,images and other objects such as interactive forms may be embedded into the rendered page.


Features:

  • Easy and simple language
  • Effective presentation
  • Platform-independent
  • Case-insensitive language

Salient Points:

  1. Filename Extension: .html
  2. Internet media type: text/html
  3. Type code: TEXT
  4. UTI(Uniform Type Identifier): public.html
  5. Developed by: World Wide Web Consortium & WHATWG
  6. Type of format: Markup Language
  7. Extended from: SGML
  8. Extended to: XHTML

Tags:

HTML Tags are like keywords which defines that how web browser will format and display the content.It mainly contains 3 parts: opening tag,content and closing tag.but some are even unclosed tags. Some of the few tags are given below...


TAG DESCRIPTION
html Declares the web page to be written in html
head Delimits the page's head
title Defines the title
body Delimits the page's body
i Set in italics
b Set in bold face
center Center on the web page
ul Unordered bulleted list
ol Ordered numbered list
br Line break
p Paragraph
hr Horizontal line
img src="....." Displays an image
a href="....." Defines a hyperlink
